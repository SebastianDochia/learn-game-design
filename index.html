GAME DESIGN

Get started with the basics of game design. Learn everything you need to get started with the adventure that is game making. Get a grasp on what makes a game fun and how to keep the player enganged for long periods of time.

Anyone can learn how to write code that lets you swing a sword. The real challange is to make that action feel enjoyable and entertaining. ~ Scott Rogers
